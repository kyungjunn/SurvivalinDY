# 🎮 Pain & Gain

> 3D 로그라이크 증강 디펜스 게임
---

## 📌 프로젝트 소개

**SurvivalinDY**는  
3D 공간에서 캐릭터를 직접 조작하며 적을 처치하고 성장하는  
**로그라이크 액션 디펜스 게임**입니다.

단순한 생존을 넘어,  
플레이 중 선택과 실패가 플레이어의 성장에 직접적인 영향을 주는  
긴장감 있는 게임 경험을 제공합니다.

---

## 🚀 주요 특징

### 🎯 3D 그래픽 기반 전투
- 현실감 있는 3D 환경에서 전투 진행
- 직접 조작(WASD) 기반 액션 플레이
- 타격감과 몰입감 강화

---

### ⚡ 성장 및 증강 시스템
- 적 처치 → 경험치 획득 → 레벨업
- 레벨업 시 **증강(Augment) 선택**
- 매 플레이마다 다른 성장 루트 제공

---

### 🔄 로그라이크 요소
- 캐릭터 사망 시 모든 능력 초기화
- 매 게임마다 새로운 플레이 경험

---

### 💀 핵심 차별점: 리스크 기반 퀘스트 시스템
- 플레이 중 랜덤 퀘스트 발생
- ❗ 퀘스트 실패 시 능력 박탈
- 전략적 선택과 긴장감 극대화

---

## 🎮 게임 플레이 흐름

1. WASD로 캐릭터 이동 및 전투
2. 적 처치 및 경험치 획득
3. 레벨업 시 증강 선택
4. 랜덤 퀘스트 수행
5. 실패 시 능력 손실 / 성공 시 추가 보상
6. 반복 플레이를 통한 다양한 빌드 경험
